## About

Reptile Cursor is a minimalist browser experiment exploring biological movement. By overriding standard pointer mechanics, it simulates fluid, organic motion through custom inverse kinematics algorithms written entirely in pure JavaScript.

## Highlights

- **Fluid Kinematics:** Multi-segmented motion simulation.
- **Dynamic Rendering:** Real-time canvas rendering.
- **Zero Dependencies:** Lightweight, vanilla implementation.
- **Optimized Performance:** High-fps logic loop.

## Setup

```bash
git clone https://github.com/YOUR_GITHUB_USERNAME/Reptile-Interactive-Cursor.git
```
Open `index.html` in any modern browser to experience the simulation.

## Core Stack

- HTML5 Canvas
- Vanilla JavaScript
- C++ (Experimental Logic)
