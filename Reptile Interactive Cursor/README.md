## ⚡ Overview

**Reptile Cursor** is a state-of-the-art browser experiment that overrides standard cursor mechanics, simulating the biological movement of a reptile via complex kinematic algorithms. Written entirely in Vanilla JS for maximum performance and zero-dependency execution. 

### ✨ Features
- 🐉 **Kinematic Skeleton:** Multi-segmented fluid movement simulation.
- 🎨 **Dynamic Rendering:** Real-time canvas drawing with optimized hardware acceleration.
- 🚀 **Zero Dependencies:** Pure, unadulterated HTML5 and Vanilla JS.
- 🌌 **Elite Aesthetics:** Futuristic styling optimized for modern digital experiences.

## 🧠 System Architecture

The core algorithm orchestrates a continuous `requestAnimationFrame` loop that calculates and updates node coordinate vectors based on distance constraints, achieving a seamlessly slithering, biomechanical entity.
